# Jayway-worksample; quiz application!

### Installation
Clone this repo
```
git clone git@github.com:theted/jayway-worksample.git
```

Install dependencies
```
npm install
```


### Running
Start the frontend server
```
npm start
```

Start the backend server
```
node server
```

### Testing
Running tests (using [jest] + [Enzyme])

```
npm run test
```


### Under the hood 
[React] is the framework used, [Stylus] is the CSS-preprocessor used. Powered by [Node.js] + [Express] <3
